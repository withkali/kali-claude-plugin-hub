virtual-book-team

v0.1.0 · MIT · withkali
5 역할 에이전트 3 슬래시 명령 6개 장르 POD · 전자책 5곳 작가 일락

책 주제를 던지면 기획자가 장르별 기획서를 잡고 → 작가 일락이 집필하고 → 북 디자이너가 완성하고 → 감수를 거쳐 → 부크크·교보 바로출판·유페이퍼·e퍼플·이페이지에 맞는 출판 파일을 준비합니다.

주요 기능

✍️

필명 일락의 집필

소설·에세이·여행기·평론까지 6개 장르를 담당하는 작가 에이전트. 장르별 문체와 구성을 구분합니다.

🎯

장르별 기획 대화

기획자가 운영자와 대화를 통해 기획서·목차·컨셉을 확정합니다. 일방적 산출이 아닌 협의 과정.

🎨

표지·내지 디자인 스펙

북 디자이너가 장르별 분위기에 맞는 표지·레이아웃·타이포그래피 가이드를 제공합니다.

📦

5개 플랫폼 출판 자동화

POD 2곳 + 전자책 3곳 규격에 맞는 파일 변환 스크립트와 업로드 단계 가이드를 생성합니다.

슬래시 명령

/new-book <장르> <주제>새 책 프로젝트 시작 — 기획 대화, 목차 확정, 집필·디자인 착수 승인 요청
/compile-essays <장르> <제목>쌓아둔 에세이·평론을 모아 한 권으로 엮는 출판 프로세스 시작
/book-roster팀 구성·장르·플랫폼 목록 출력

지원 장르

#장르기획 포커스
1📖 일반 소설 (드라마)주제·인물 설정·플롯 흐름
2🕵️ 장르 소설추리/스릴러/의학/법정/판타지/SF — 현실고증 필수
3💡 기술 서적 / 리더십 / 조직론강의식 구성·목차·박스 지식 배치
4🌿 에세이짧은 주제 + 일화 → 맛깔스러운 구성, 컬렉션 출판 지원
5✈️ 여행기사진·일정 기반, 사람 중심 이야기
6🎬 영화/드라마/책 평론핵심 씬·대사 + 평론가의 감성 해석

🔍 reviewer (감수) opus

출판 플랫폼

🟦

부크크

국내 대표 POD. 종이책 주문 제작·유통 연계 가능.

🟩

교보문고 바로출판

교보 직계 POD. 교보 서점 등록 가능.

📱

유페이퍼

ePub/PDF 지원 전자책 플랫폼. 작가 직접 판매.

📗

e퍼플

한국출판콘텐츠 플랫폼. ePub 형식 필수.

📘

이페이지

PDF/ePub 지원. 간편 업로드 방식.

워크플로우

  • Phase 1 · 기획 — 주제·장르 인입 → 기획자·운영자 대화 → 목차 확정 → 작가 사전 검토 → 운영자 승인
  • Phase 2 · 집필 & 디자인 — 작가 챕터별 집필 → 북 디자이너 표지·내지 스펙 → 감수 → 운영자 최종 승인
  • Phase 3 · 출판 — 플랫폼별 파일 변환 → 업로드 가이드 제공 → 운영자가 직접 업로드

에세이 · 평론 컬렉션 워크플로우

  • 에세이와 평론은 단편씩 쌓다가 나중에 모아 출판합니다.
  • /compile-essays로 기존 글을 정리·순서 결정 → 필요 시 추가 집필 → 감수 → 출판.
  • 이미 발행된 글은 저작권·독점 여부를 먼저 확인합니다.

보안 · 자율성

  • 로컬 파일 작업(기획·집필·디자인 스펙)은 자동 진행.
  • 플랫폼 업로드·계정 접근은 운영자 직접 수행 — 자동화 범위 밖.
  • 저작권 있는 텍스트 무단 인용 금지. 평론의 인용은 공정 이용 범위 내.
  • 개인정보(실명·결제정보) 파일 저장 금지.
여행기 집필을 시작하려면 사진 목록과 여행 일정을 먼저 제공해주세요. 기획자가 에피소드 단위로 구성을 잡습니다.
