Claude Code용 플러그인 · 스킬 · MCP를 한 곳에서 받아 설정합니다.

Marketplace: withkali/kali-claude-plugin-hub

1. 마켓플레이스 추가

Claude Code에서 한 번만 추가하면 됩니다.

/plugin marketplace add withkali/kali-claude-plugin-hub

CLI로도 가능: claude plugin marketplace add withkali/kali-claude-plugin-hub

2. 플러그인

virtual-team v0.1.0

기획자·디자이너·클라이언트(iOS/Android/크로스)·웹(UI/프론트)·서버(DBA/API)·인프라·테스터·배포·장애관리 22개 역할의 서브에이전트로 서비스를 기획→개발→(운영)까지 자동화하는 가상 개발팀. 시크릿 하드코딩 차단 훅과 오케스트레이션 슬래시 명령 포함.

/plugin install virtual-team@kali-claude-plugin-hub

설치 후: /new-service <아이디어> · /team-roster · /activate-phase3 <slug>

👉 virtual-team 가이드 문서

3. 스킬

스킬은 플러그인에 동봉되거나 단독 배포될 수 있습니다. 현재 이 허브에는 단독 스킬이 없습니다(추가 시 여기 표시).

설치한 플러그인이 제공하는 스킬은 자동으로 사용 가능해지며, /help나 스킬 목록에서 확인할 수 있습니다.

4. MCP 서버

일부 플러그인은 MCP 서버를 동봉합니다. 단독 MCP가 필요하면 프로젝트 또는 사용자 범위로 추가합니다.

# 예: 사용자 범위로 MCP 서버 추가 (stdio)
claude mcp add <name> -- <command> [args...]

# 등록된 MCP 확인
claude mcp list

virtual-team은 별도 MCP가 필요 없습니다 — 내장 도구와 gh CLI로 동작합니다.

5. 권한(자동화) 설정

플러그인은 권한 allow/deny 목록을 담지 못합니다. 확인 프롬프트를 줄이려면 ~/.claude/settings.json(전역) 또는 프로젝트 .claude/settings.json에 권한을 둡니다. 권장(균형): 로컬·가역 작업은 자동 허용, 원격 push·배포·prod 변경·시크릿은 확인.
